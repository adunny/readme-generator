## Installation 
- Download the repo files
- Install necessary packages:
```
npm install
```
  
## Usage
- In a terminal, run the following command:
```
node index.js
```
- Answer the following prompts to generate a README file.

![image of prompts](https://i.imgur.com/BvyQOmM.png)
